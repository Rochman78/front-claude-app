FORMAT JSON — CRÉATION DE DEVIS PENNYLANE
==========================================

Ce document décrit comment formater le JSON à fournir au gérant pour la création du devis PDF dans Pennylane.
Une fois le chiffrage validé par le client, tu dois formater ces données au format JSON ci-dessous.

QUAND FORMATER LE JSON :
- Le client a validé le chiffrage (mail texte) que tu lui as envoyé
- Le gérant te confirme que le client a validé
- Tu disposes de toutes les infos nécessaires (voir checklist ci-dessous)

CHECKLIST AVANT DE FORMATER :
□ Code boutique identifié
□ Nom/prénom ou raison sociale du client
□ Email du client
□ Adresse complète (rue, code postal, ville)
□ Adresse de livraison (si différente)
□ Produit(s) : label + quantité + prix unitaire HT + unité
□ Accessoires si applicable
□ Transport : inclure les lignes si mentionné dans le chiffrage


CODES BOUTIQUE :
- LFC → Le Filet de Camouflage
- LVO → Le Voile d'Ombrage
- COCO → Ma Toile Coco
- MON → Mon Ombrage
- TAR → Tarnnetz
- HET → Het Camouflagenet
- RED → Red de Camuflaje
- RETE → Rete Mimetica
- UNI → Universel


STRUCTURE JSON — CLIENT PARTICULIER :

{
  "store": "[CODE BOUTIQUE]",
  "customer": {
    "type": "individual",
    "firstName": "[PRÉNOM]",
    "lastName": "[NOM]",
    "email": "[EMAIL]",
    "phone": "[TÉLÉPHONE ou vide]",
    "address": {
      "address": "[RUE]",
      "postalCode": "[CODE POSTAL]",
      "city": "[VILLE]"
    }
  },
  "subject": "[OBJET DU DEVIS — description courte]",
  "lines": [
    {
      "type": "product",
      "label": "[DESCRIPTION PRODUIT]",
      "description": "[DÉTAILS : quantité, surface, délai...]",
      "quantity": [NOMBRE],
      "unitPrice": "[PRIX HT en string]",
      "unit": "m2"
    }
  ]
}


STRUCTURE JSON — CLIENT PROFESSIONNEL :

{
  "store": "[CODE BOUTIQUE]",
  "customer": {
    "type": "company",
    "name": "[RAISON SOCIALE]",
    "firstName": "[PRÉNOM CONTACT]",
    "lastName": "[NOM CONTACT]",
    "email": "[EMAIL]",
    "phone": "[TÉLÉPHONE]",
    "vatNumber": "[N° TVA INTRA si applicable]",
    "address": {
      "address": "[RUE]",
      "postalCode": "[CODE POSTAL]",
      "city": "[VILLE]"
    }
  },
  "subject": "[OBJET DU DEVIS]",
  "lines": [ ... ]
}


ADRESSE DE LIVRAISON DIFFÉRENTE :
Ajouter dans customer :

    "deliveryAddress": {
      "address": "[RUE LIVRAISON]",
      "postalCode": "[CODE POSTAL]",
      "city": "[VILLE LIVRAISON]"
    }


TYPES DE LIGNES (lines) :

"product" — Produit principal (filet, voile, toile)
  quantity : nombre de m² ou pièces (NOMBRE, pas de guillemets)
  unitPrice : prix HT (STRING entre guillemets, ex : "12.50")
  unit : "m2" pour produits au m², "piece" pour produits à l'unité

"accessory" — Accessoire (mât, corde, kit fixation, câble acier)
  quantity : nombre d'unités
  unitPrice : prix HT en string
  unit : "piece" (par défaut)

"transport" — Frais de livraison
  unitPrice : "19.99" (standard)

"transport_discount" — Remise livraison (livraison offerte)
  unitPrice : "-19.99"

"free" — Ligne gratuite (échantillon, cadeau)
  unitPrice : "0.00"


RÈGLES IMPORTANTES :
- unitPrice est TOUJOURS un string entre guillemets : "45.00" (pas 45.00)
- quantity est TOUJOURS un nombre SANS guillemets : 1.43 (pas "1.43")
- Le point est le séparateur décimal (pas la virgule) : "12.50" (pas "12,50")
- Si livraison offerte : TOUJOURS inclure les 2 lignes transport + transport_discount
- Une ligne par accessoire (ne pas regrouper)
- Le subject est l'objet affiché sur le PDF du devis


EXEMPLE COMPLET :

Chiffrage validé :
"Client : Mathieu Bajard, matthieu.bajard@gmail.com, 06 66 89 31 38
Adresse : 26 impasse de l'amendier, 05230 Chorges
2 filets polyester sable 0.75 x 0.95 m
Surface totale : 1.43 m²
Prix : 45,00 € HT/m²
Livraison offerte"

JSON à fournir au gérant :

{
  "store": "LFC",
  "customer": {
    "type": "individual",
    "firstName": "Mathieu",
    "lastName": "BAJARD",
    "email": "matthieu.bajard@gmail.com",
    "phone": "0666893138",
    "address": {
      "address": "26 impasse de l'amendier",
      "postalCode": "05230",
      "city": "Chorges"
    }
  },
  "subject": "Devis filet polyester sable 0.75x0.95m",
  "lines": [
    {
      "type": "product",
      "label": "SABLE - 0.75x0.95 m - Filet de camouflage renforcé",
      "description": "Quantité : 2 | Total m² : 1.43 | Délai : ~14 jours",
      "quantity": 1.43,
      "unitPrice": "45.00",
      "unit": "m2"
    },
    {
      "type": "transport",
      "label": "Transport sur mesure",
      "unitPrice": "19.99"
    },
    {
      "type": "transport_discount",
      "label": "Remise transport sur mesure",
      "unitPrice": "-19.99"
    }
  ]
}

COMMENT PRÉSENTER LE JSON AU GÉRANT :
Quand tu as formaté le JSON, présente-le au gérant ainsi :

"Voici le JSON à coller dans l'outil de création de devis :"

[le JSON]

"Une fois le devis créé, tu pourras envoyer le mail avec le lien PDF directement depuis l'outil."
